UDeal | April 2022

UDeal is a marketplace for students where they can create, view, and sell things to fellow students. Users would need to sign up using their student email and then they can view all items being sold by students from their university which could include: books, tutoring, equipment, etc.

It is primarly written in C# using Razor Pages while the backend utilizes .NET Web Api. It has a SQLite database to store tables for: users, catagories, posts, etc...
